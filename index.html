Stata Analysis: Vitamin A Impact on Child Health

Stata Data Visualization Health Statistics FSNAU Data

This Stata script analyzes the relationship between Vitamin A supplementation coverage and child health outcomes in Somalia, revealing a powerful inverse correlation between Vitamin A coverage and both morbidity and malnutrition rates. The analysis uses FSNAU data from 2015-2024 to generate publication-ready visualizations for evidence-based policy recommendations.

clear all
set more off

*---------------------------------------------------------------*
* 0. PATHS
*---------------------------------------------------------------*
global ROOT "~/nutrition_analysis/dashboard"
global DATA "$ROOT/fsnau_nutrition_2009_2024.csv"
global OUT "$ROOT/graphs"
capture mkdir "$OUT"

set scheme s1mono

*---------------------------------------------------------------*
* 1. DATA LOAD & CLEAN
*---------------------------------------------------------------*
import delimited using "$DATA", clear varn(1)
rename *, lower

foreach var in gam vitamina morbidity year {
    capture confirm numeric variable `var'
    if _rc destring `var', replace force
}

label var gam "GAM (%)"
label var vitamina "Vitamin A Coverage (%)"
label var morbidity "Morbidity (%)"

keep year gam vitamina morbidity
drop if missing(year)
keep if year >= 2015

collapse (mean) gam vitamina morbidity, by(year)

*---------------------------------------------------------------*
* 2. DEFINE CORRELATION YEARS
*---------------------------------------------------------------*
local corr_years 2018 2019 2022

foreach y of local corr_years {
    quietly summarize gam if year==`y'
    local gam_`y' = r(mean)
    quietly summarize morbidity if year==`y'
    local morbidity_`y' = r(mean)
    quietly summarize vitamina if year==`y'
    local vita_`y' = r(mean)
}

*---------------------------------------------------------------*
* 3. ROUND LABELS FOR CLEAN DISPLAY
*---------------------------------------------------------------*
gen gam_round = round(gam)
gen morbidity_round = round(morbidity)
gen vitamina_round = round(vitamina)

*---------------------------------------------------------------*
* 4. GRAPH: Dual-Axis with Matching Colors
*---------------------------------------------------------------*
set scheme s1color  // Ensures legend matches plot colors

twoway ///
    (line gam year, lcolor("30 136 229") lwidth(medthick) lpattern(solid) yaxis(1)) ///
    (line morbidity year, lcolor("255 167 38") lwidth(medthick) lpattern(dash) yaxis(1)) ///
    (line vitamina year, lcolor("67 160 71") lwidth(medthick) lpattern(shortdash_dot) yaxis(2)) ///
    (scatter gam year if inlist(year,2018,2019,2022), msymbol(O) mcolor("30 136 229") msize(medlarge) ///
        mlabel(gam_round) mlabsize(small) mlabposition(14)) ///
    (scatter morbidity year if inlist(year,2018,2019,2022), msymbol(D) mcolor("255 167 38") msize(medlarge) ///
        mlabel(morbidity_round) mlabsize(small) mlabposition(6)) ///
    (scatter vitamina year if inlist(year,2018,2019,2022), msymbol(T) mcolor("67 160 71") msize(medlarge) ///
        mlabel(vitamina_round) mlabsize(small) mlabposition(11)) ///
    , ///
    title("Impact of Vitamin A Coverage on Child Health", size(large) margin(b=5)) ///
    subtitle("Trends in Morbidity and Acute Malnutrition, Somalia 2015-2024", size(medsmall)) ///
    xtitle("Year", size(small)) ///
    ytitle("Prevalence (%)", axis(1) size(small)) ///
    ytitle("Vitamin A Coverage (%)", axis(2) size(small)) ///
    xlabel(2015(1)2024, angle(45) labsize(small)) ///
    ylabel(0(5)30, axis(1) labsize(small) grid glcolor(gs14) glwidth(vthin)) ///
    ylabel(50(10)90, axis(2) labsize(small) grid glcolor(gs14) glwidth(vthin)) ///
    legend(order(1 "Acute Malnutrition (GAM)" 2 "Morbidity Rate" 3 "Vitamin A Coverage") ///
        pos(6) row(1) size(small) region(lcolor(none)) colfirst symplacement(center)) ///
    graphregion(color(white)) plotregion(color(white)) bgcolor(white) aspectratio(0.6) ///
    note("Source: FSNAU Somalia Data. Markers show key years demonstrating the inverse relationship.", size(vsmall))

graph export "$OUT/vitA_trends.png", replace width(2000)

Statistical visualization showing inverse relationship between Vitamin A coverage and malnutrition/morbidity rates

Primary Finding

Inverse relationship between Vitamin A coverage and child health outcomes — When coverage drops, both morbidity and malnutrition increase significantly. This demonstrates the critical importance of sustained micronutrient coverage programs.

Policy Implication

Targeted micronutrient programs are non-negotiable components of global health strategy, with measurable impact on population health indicators. Evidence-based resource allocation can maximize health gains.
